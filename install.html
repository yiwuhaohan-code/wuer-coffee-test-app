無二咖啡

门店操作 App

打开 App

让店员或家人扫码打开。打开后按下面步骤添加到手机桌面，以后从桌面图标进入。

iPhone 安装

  1. 用 Safari 打开本页，不要直接在微信内安装。
  2. 点底部分享按钮。
  3. 选择 添加到主屏幕。
  4. 名称保持 無二咖啡，点添加。

安卓安装

  1. 用 Chrome 或系统浏览器打开本页。
  2. 点右上角菜单。
  3. 选择 安装应用 或 添加到主屏幕。
  4. 安装后从手机桌面的無二咖啡图标进入。

数据注意

  1. 数据保存在当前手机浏览器里，换手机前先在 App 里做备份。
  2. 不要随便清除浏览器数据，否则本机记录可能丢失。
  3. 后续我更新 App，她重新打开就会自动更新；如果没更新，重新扫二维码打开一次。
